**new tests needed:**

1. **size level defaults**
   - given no --size, creates medi-level stones
   - given --size=nano, creates only nano-level stones
   - given --size=giga, creates all stones (same as mega)

2. **size + guard composition**
   - given --size=nano --guard=heavy, creates nano stones with heavy guards
   - given --size=mega --guard=light, creates mega stones with light guards

3. **size-specific stone presence**
   - nano: verify only wish, vision, blueprint, roadmap, execution
   - mini: verify nano + criteria, code research, evaluation
   - medi: verify mini + reflection, playtest, repros
   - mega: verify medi + domain research, factory research, distillation, factory blueprint
   - giga: verify same as mega

4. **size-specific stone absence**
   - nano: verify NO criteria, NO research, NO playtest, NO evaluation
   - mini: verify NO reflection, NO playtest, NO factory research
