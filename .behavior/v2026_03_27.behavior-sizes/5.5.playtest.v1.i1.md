# playtest: behavior sizes

## prerequisites

- this repo cloned and on feature branch
- pnpm installed
- node 18+

## sandbox

all commands target `.temp/` directory which is gitignored.

## happy paths

### 1. verify --size nano

```bash
cd .temp && rm -rf test-nano && mkdir test-nano && cd test-nano
git init && git commit --allow-empty -m "init"
git checkout -b feature/nano-test
```

```bash
npx tsx ../../../bin/run skill init.behavior --name nano-test --size nano
```

**expected:**
- output shows `oh, behave!` with ~9 files
- NO criteria files (2.x)
- NO research files (3.x)
- YES wish, vision, blueprint, roadmap, execution

```bash
ls -la .behavior/*/
```

**verify:**
- `0.wish.md` extant
- `1.vision.stone` extant
- `2.1.criteria.blackbox.stone` NOT extant
- `3.1.*.stone` NOT extant

## pass/fail criteria

- pass if: all 5 happy paths show expected file counts and file presence
- pass if: size + guard compose independently
- pass if: invalid size value throws zod error
- pass if: giga produces same output as mega
- fail if: any size produces wrong templates
- fail if: default size is not medi
- fail if: size and guard interfere with each other
