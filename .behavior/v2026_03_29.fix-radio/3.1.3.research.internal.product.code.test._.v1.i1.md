## 1. jest environment patterns

### [EXTEND] jest.integration.env.ts

current implementation validates prerequisites before test execution [1]:

```typescript
// eslint-disable-next-line no-undef
jest.setTimeout(90000); // since we call downstream apis

// verify package.json exists
if (!existsSync(join(process.cwd(), 'package.json')))
  throw new Error('no package.json found in cwd. are you @gitroot?');

// verify test environment
if (
  (process.env.NODE_ENV !== 'test' ||
    (process.env.STAGE && process.env.STAGE !== 'test')) &&
  process.env.I_KNOW_WHAT_IM_DOING !== 'true'
)
  throw new Error(`integration.test targets stage other than 'test'`);

// verify aws auth if needed
const requiresAwsAuth = declapractUseContent.includes('awsAccountId');
if (requiresAwsAuth && !(process.env.AWS_PROFILE || process.env.AWS_ACCESS_KEY_ID))
  throw new Error('no aws credentials present...');
```

**extension needed**: add keyrack.source() call per xai pattern [2].

proposed addition:

```typescript
import { keyrack } from 'rhachet/keyrack';

const keyrackYmlPath = join(process.cwd(), '.agent/keyrack.yml');
if (existsSync(keyrackYmlPath))
  keyrack.source({ env: 'test', owner: 'ehmpath', mode: 'strict' });
```

### [EXTEND] jest.acceptance.env.ts

current implementation validates aws auth [1]:

jest.setTimeout(90000);

## 4. integration test patterns

### [REUSE] getGithubTokenByAuthArg.integration.test.ts

current tests validate shx pattern [6]:

```typescript
given('[case1] --auth as-robot:shx(echo token)', () => {
  when('[t0] shell command outputs token', () => {
    then('returns token from command output', async () => {
      const result = await getGithubTokenByAuthArg(
        { auth: 'as-robot:shx(echo secret-from-echo)' },
        { env: {}, shx },
      );
      expect(result.token).toBe('secret-from-echo');
      expect(result.role).toBe('as-robot');
    });
  });
});
```

pattern is correct. extend with keyrack integration tests.

### [NEW] keyrack integration tests

needed tests:

```typescript
given('[case] keyrack provides token', () => {
  when('[t0] keyrack has EHMPATH_BEAVER_GITHUB_TOKEN', () => {
    then('returns keyrack token', async () => {
      const result = await getGithubTokenByAuthArg(
        { auth: undefined },
        { env: {}, shx },
      );
      expect(result.token).toMatch(/^ghs_/);
      expect(result.role).toBe('as-robot');
    });
  });
});
```
