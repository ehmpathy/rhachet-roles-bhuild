**rule**: .agent/repo=bhuild/role=behaver/briefs/practices/behavior.execution/architect/rule.forbid.maintenance-hazards.md

**locations**:

Variables are declared with 'let' and mutated via assignment after declaration (e.g. to capture exids from async results). This violates the immutability requirement (use const + spreads/returns instead of let + assignment). Creates shared mutable state that is hard to reason about and leads to debug hours.
